# ComputeAsset

Used to describe algorithms when initializing new compute jobs via [`nautilus.compute()`](/docs/api/nautilus/compute)

## Usage
This type is used for the `algorithm` properties when starting compute jobs.

```ts twoslash
import { Nautilus } from '@deltadao/nautilus'
import { Wallet } from 'ethers'
const nautilus = await Nautilus.create(new Wallet('0x1234'))
// ---cut---
await nautilus.compute({
    dataset: { 
        did: 'did:op:1234abcd...',
    }, 
    algorithm: { // [!code focus]
        did: 'did:op:5678wxyz...' // [!code focus]
    } // [!code focus]
})
```

## Properties

This interface inherits the same properties provided by [`ComputeAsset`](/docs/api/ComputeAsset).

In addition, it provides the following properties.

### algocustomdata (optional)
- Type: [`UserCustomParameters`](/docs/api/UserCustomParameters)

The decentralized identifier of the asset to compute on.
```ts twoslash
import { ComputeAsset } from '@deltadao/nautilus'
// ---cut---
const dataset = {
    did: 'did:op:1234abcd...' // [!code focus]
} satisfies ComputeAsset
```

### serviceId (optional)
- Type: `string`

Optionally specify a service to use for computation. Defaults to the first compute service of the asset's metadata.
```ts twoslash
import { ComputeAsset } from '@deltadao/nautilus'
// ---cut---
const dataset = {
    did: 'did:op:1234abcd...', 
    serviceId: 'specific-service-id', // [!code focus]
} satisfies ComputeAsset
```

### userdata (optional)
- Type: [`UserCustomParameters`](/docs/api/UserCustomParameters)

If the asset requires or allows for custom user parameter input, this can be configured here.

Learn more about how to use custom userdata with your assets in our [Consumer Parameters Guide](/docs/guide/consumer-parameters).
```ts twoslash
import { ComputeAsset } from '@deltadao/nautilus'
// ---cut---
const dataset = {
    did: 'did:op:1234abcd...', 
    serviceId: 'specific-service-id',
    userdata: { // [!code focus]
        parameter: 'value', // [!code focus]
        number: 123, // [!code focus]
        boolean: true, // [!code focus]
        select: 'option', // [!code focus]
    } // [!code focus]
} satisfies ComputeAsset
```
