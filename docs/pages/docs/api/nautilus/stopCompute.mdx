# stopCompute

Send a stop request for a given compute job hosted by a specific provider.

## Usage

```ts twoslash
import { Nautilus } from '@deltadao/nautilus'
import { Wallet } from 'ethers'

const nautilus = await Nautilus.create(new Wallet('0x1234'))

await nautilus.stopCompute({ // [!code focus]
    did: 'did:op:1234abcd...', // [!code focus]
    jobId: 'abcd1234', // [!code focus]
    providerUri: 'https://provider.dev.pontus-x.eu' // [!code focus]
}) // [!code focus]
```

## Returns

[`ComputeJob | ComputeJob[]`](https://github.com/oceanprotocol/ocean.js/blob/main/src/%40types/Compute.ts#L34-L47) {/*link to type in docs if possible*/}

The compute job object containing information about the job status.

## Parameters 

### config
- Type: `ComputeStopConfig`

The configuration object to request the stop of a compute job.
