# setLogLevel

Set the level for logs of all nautilus instances.

## Usage

```ts twoslash
import { LogLevel, Nautilus } from '@deltadao/nautilus'

const nautilus = await Nautilus.setLogLevel(LogLevel.Verbose) // [!code focus]
```

## Returns

`void` {/*link to type in docs if possible*/}

## Parameters

### level
